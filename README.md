## Leadway Assessment API

## Project setup

```bash
$ pnpm install

# Make sure to set up your environment variables
$ cp .env.example .env
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
